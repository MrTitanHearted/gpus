# gpus

--------------------------------------

## Arguments

### "--backend"

* vulkan
* dx12
* dx11
* metal
* gl
* web_gpu
* primary: vulkan | dx12 | metal | web_gpu
* secondary: dx11 | gl

--------------------------------------

### "--features"

> Shows gpus' features.

--------------------------------------

### "--limits"

> Shows gpus' limits.

### "--device_type"

* dedicated-gpu
* integrated-gpu
* cpu
* software | virtual-gpu
